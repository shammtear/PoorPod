# PoorPod
Poor's Pod is a launcher and a music player for Android

# Requirements
- Android Pie(9) or higher

# Permissions
 - READ_PHONE_STATE
 - ACCESS_WIFI_STATE
 - BLUETOOTH_ADMIN
 - BLUETOOTH
 - VIBRATE
 - BATTERY_STATS
 - WRITE_SETTINGS
 - WRITE_EXTERNAL_STORAGE
 - MODIFY_AUDIO_SETTINGS

 Also, you can read the Android Manifest too :)
